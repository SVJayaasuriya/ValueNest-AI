# ValueNest AI

ValueNest AI is a machine learning based project that predicts the estimated cost of building a house in different cities in India. It calculates land cost, material cost, and labor cost based on user inputs and provides a detailed construction budget.

## Features
- Predict house construction cost  
- Supports multiple states and cities  
- Calculates land, materials, and labor costs  
- Interior type selection (Simple / Modern / Futuristic)  
- Detailed cost breakdown  

## Tech Stack
- Python  
- Flask  
- Machine Learning  
- HTML/CSS/JS  

## How to Run

1. Clone the repository:
   git clone https://github.com/SVJayaasuriya/ValueNest-AI.git  
   cd ValueNest-AI  

2. Install dependencies:
   pip install requirements.txt 

3. Run the app:
   python app.py  

4. Open in browser:
   http://127.0.0.1:5000/

## Goal
To help users estimate and plan house construction budgets using AI.
